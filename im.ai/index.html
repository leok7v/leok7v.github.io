Features

Everything you need from AI

A thoughtfully designed assistant for privacy-conscious users and AI enthusiasts alike.

Local Models — Privacy First

Run AI models directly on your device using llama.cpp. All conversations stay completely offline.

  • Llama, Gemma, Qwen, Falcon & more
  • No internet required
  • Automatic download with progress tracking

Cloud Providers

Connect to leading AI providers using your own API keys — no middlemen, no markups.

  • Google Gemini, Groq, OpenRouter
  • Cerebras, GitHub Models
  • Switch providers on the fly

Beautiful Chat UI

Gorgeous markdown rendering with code syntax highlighting, session history, and instant model switching.

  • Full markdown & code blocks
  • Session management & search
  • Conversation context display

Optimized Performance

Low memory footprint and hardware-accelerated inference. Lightning-fast whether local or cloud.

Customizable Experience

Adjust text size and appearance. Securely store API keys. Organise conversations your way.

Completely Free
